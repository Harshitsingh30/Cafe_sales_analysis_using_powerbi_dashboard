Project overview

1. Project Title: 
Cafe Sales Analysis Using Power BI Dashboard. 

2. Objective: 
The main objective of this project is to analyze cafe sales data and identify sales patterns, 
customer preferences, and business performance trends. The dashboard helps in making 
data-driven decisions to improve revenue and customer satisfaction.
 
3. Dataset Source: 
The dataset used for this project was sourced from Kaggle.

4. Tools Used: 
Microsoft Excel, 
Microsoft Power BI,
Power Query Editor

5. Project Process: 
A. Excel: 
Performed initial data understanding and explored the dataset. 
Created a short summary of the dataset.

B. Power BI: 
Imported the dataset into Power BI. 
Performed data cleaning using Power Query Editor. 
Executed the ETL (Extract, Transform, Load) process. 
Enhanced the dataset by creating additional columns such as: 
Month Name 
Month Number 
Day Name 
Day Number 
Total Spent 
Created DAX measures for business analysis. 
Designed interactive visualizations and dashboards. 
Conducted data analysis using the Power BI dashboard.

6. Key Insights:
(1) Although Coffee received the highest number of orders, Salad generated the highest 
revenue. 
(2) Cookies recorded the lowest number of orders and generated the least revenue.
(3) The highest number of orders was recorded in October, while the lowest number of orders 
 was recorded in February. 
(4) Monday was identified as the busiest day compared to other days of the week. 
(5) Customers slightly preferred Digital Wallets over other payment methods.
    
7. Recommendations: 
Promote high-revenue items such as Salad through special offers and prominent menu 
placement to maximize revenue. 
Introduce combo offers or discounts to improve the sales of low-performing items such as 
Cookies. 
Launch promotional campaigns and discounts during February to increase sales during 
low-demand periods. 
Ensure adequate staffing and inventory management on Mondays, as it is the busiest day of 
the week. 
Maintain smooth and reliable digital payment facilities to enhance customer satisfaction and 
improve customer experience.

8. Conclusion: 
The Cafe Sales Analysis Dashboard provides valuable insights into customer behavior, sales 
trends, and product performance. These insights can help cafe management optimize 
operations, increase revenue, and improve overall customer satisfaction.
